### Keyboard shortcuts

An exhaustive list of available keyboard shortcuts can be access by pressing `?`, or by clicking the question mark icon.

| Key      | Action |
| ----------- | ----------- |
| `J` / `L`      | Previous / next map       |
| `I` / `K`   | Previous / next individual        |
| `N`   | Create new pane        |
| `O`   | Toggle map description        |
| `U`   | Toggle mean across individuals        |
| `S`   | Lateral view        |
| `F`   | Medial view        |
| `E`   | Anterior view        |
| `D`   | Posterior view        |
